# Iris Clustering Lab

This project applies k-means and hierarchical clustering to the Iris dataset
using petal length and petal width as features.

## Methods
- K-means clustering (k=3)
- Agglomerative hierarchical clustering
- Euclidean and cityblock distances

## Libraries
- scikit-learn
- matplotlib
- scipy
- numpy

## How to Run
```bash
pip install -r requirements.txt
python src/clustering.py
